# Fix Agent Team Messages Displayed as "You" in Conversation View

**Goal:** Detect these agent messages in the rendering layer and display them with a distinct visual treatment â€” no re-sync needed, no schema changes.

## Approach

Rendering-layer-only fix in the dashboard. Detection via content pattern matching in `preprocess.ts`, delegation to a new `AgentMessageBubble` component from `MessageBubble`.

- `isAgentMessage(content: string): boolean` â€” lightweight check using `string.includes()` (no regex, fast path for 99% of messages)

Patterns to detect:

- `<teammate-message teammate_id="..." color="...">` â€” extract `teammateId`, `color`, inner JSON (`type`, `from`, `summary`)
