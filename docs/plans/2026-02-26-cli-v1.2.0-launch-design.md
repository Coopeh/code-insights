## Context

The CLI (`@code-insights/cli`) is at v1.2.0 but has never been published to npm. Git tags exist for v1.0.0â€“v1.0.2. Since v1.0.x, significant features were added:

- **Stats command suite** (5 subcommands: dashboard, cost, projects, today, models)
- **Multi-tool support** (Cursor, Codex CLI, Copilot CLI providers)
- **Zero-config mode** (local-first, no Firebase required for stats)
- **Telemetry** (anonymous, opt-out, fire-and-forget)
- **`open` command** (launch web dashboard in browser)

Current state: 0 test coverage, 0 ESLint config, documented Windows path bugs, no CHANGELOG.

## Phase 1: QA Test Matrix

### Provider Discovery & Parse

| Provider | Test | Verification |
|----------|------|-------------|
| Claude Code | Discovers sessions in `~/.claude/projects/` | `sync --dry-run` shows sessions |
| Claude Code | Parses JSONL (messages, tool calls, timestamps) | Sync + verify in Firestore |
| Claude Code | Smart title generation | Check `generatedTitle` + `titleSource` |
| Cursor | Discovers `.vscdb` in workspace storage | `sync --source cursor --dry-run` |
| Cursor | Parses SQLite tabs | Verify message content |
| Codex CLI | Discovers Codex session files | `sync --source codex-cli --dry-run` |
| Codex CLI | Parses event-stream JSONL | Verify message structure |
| Copilot CLI | Discovers Copilot session files | `sync --source copilot-cli --dry-run` |
| Copilot CLI | Parses event JSONL | Verify message structure |

### Command Tests

| Command | Test Case | Expected |
|---------|-----------|----------|
| `init` | Fresh setup (no config) | Wizard completes, config saved |
| `init --from-json` | Import service account | Config imported, validated |
| `sync` | Normal sync | Sessions uploaded |
| `sync --dry-run` | Dry run | Shows what would sync, no writes |
| `sync --force` | Force re-sync | Re-processes all |
| `sync -q` | Quiet mode | Minimal output |
| `status` | Show state | Config, last sync, data source |
| `stats` | Dashboard | Terminal chart renders |
| `stats cost` | Cost breakdown | Model costs by project |
| `stats projects` | Project cards | Projects with sparklines |
| `stats today` | Today's sessions | Today's activity |
| `stats models` | Model distribution | Model usage chart |
| `config` | View config | Prints config |
| `config set-source local` | Switch to local | Source updates |
| `config set-source firebase` | Switch to firebase | Source updates |
| `connect` | Generate URL | Base64-encoded URL output |
| `open` | Launch dashboard | Browser opens |
| `install-hook` | Install auto-sync | Hook in Claude Code config |
| `uninstall-hook` | Remove hook | Hook removed |
| `reset --confirm` | Delete data | Firestore cleared |
| `telemetry status` | Check state | Shows enabled/disabled |
| `telemetry disable` | Opt out | Disabled |

### Edge Cases

| Scenario | Expected |
|----------|----------|
| No config, run `sync` | Error: "Run `code-insights init` first" |
| No sessions found | Graceful message, no crash |
| Invalid Firebase credentials | Clear error |
| `--help` on every command | Help renders |
| `code-insights` (no args) | Shows help/usage |
